## 1) 评分规则（用于筛选）

- `授权清晰度`（1-5）：是否能明确判断可商用边界
- `音频质量`（1-5）：素材质量、录制/混音稳定性
- `冥想适配度`（1-5）：是否适合减压/睡眠/专注/恢复场景
- `接入成本`（1-5）：前端接入难度（5=最易接入）
- `综合评分`：四项平均值（四舍五入到 0.5）

> 说明：本清单为产品实现调研，不构成法律意见。上线前需保留许可证快照与下载凭证（时间戳、资源 ID、license 版本）。

## 4) 结论与推荐（给 T-803/T-804）

### 推荐优先集（先落地，风险最低）

- 音频主源：`A01 Pixabay` + `A03 Mixkit` + `A05 Incompetech`
- 补充源：`A09 Freesound`（仅 `CC0/CC BY`）+ `A11 Wikimedia`（逐文件核验）
- 交互基线：`I01 Insight Timer` + `I04 Headspace` + `I05 Balance`

### 两套可落地方案

#### 方案 A（保守版，低成本快速上线）

- 资源池：`A01 + A03 + A05 + A09(仅CC0/CC BY)`
- 目标：先交付 12 首可用冥想音频（减压/睡眠/专注/恢复各 3 首）
- 授权策略：
  - 每条素材保留 `source_url + license_type + downloaded_at + attribution_text`
  - 禁止纳入 `NC` 与授权不明素材
- 成本：接近 0（主要为人工筛选成本）
- 风险：内容同质化偏高、品牌独特性一般

#### 方案 B（增强版，品质优先）

- 资源池：`A18 Artlist + A19 Epidemic + A22 Pond5`（必要时 `A20/A21` 补位）
- 目标：构建 20-30 首高质量曲库（含主题封面、时长、标签、版权元数据）
- 授权策略：
  - 在素材元数据中固化 `plan_tier + project_scope + license_snapshot`
  - 订阅型资源要求“发布前二次校验订阅状态”
- 成本：中高（订阅或单曲采购）
- 收益：音质和风格一致性更强，商业可用稳定性更高

## 5) 执行清单（可直接衔接 T-803）

- 从 A 方案先筛出 12 首“可立即接入”曲目（覆盖 4 类场景）。

- 不同平台条款会更新，发布前需复核当日条款。

- 本项目定位为心理教练工具，避免“医疗诊断/治疗承诺”表达。
